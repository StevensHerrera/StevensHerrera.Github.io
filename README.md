## How to Run

**Option 1 — View online:**

1. Open your browser and paste this link in the address bar:
   ```
   https://stevensherrera.github.io/
   ```

**Option 2 — Clone locally:**

1. Clone the repository:
   ```bash
   git clone https://github.com/StevensHerrera/StevensHerrera.Github.io.git
   ```

2. Open `index.html` in your browser, or use an extension like **Live Server** in VS Code to run it with a local server.

    > No dependencies or additional setup required.
